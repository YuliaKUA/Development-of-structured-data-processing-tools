# Development-of-structured-data-processing-tools
Coursework in the discipline Object-Oriented Programming. 

The paper shows an example of class development and the processing of structured data. 
It is necessary to determine the "maximum object" from the input information and write it to a new resulting file. 
Process the rest of the data. 
During the execution of the program, processing and initialization of exceptional situations is provided.
